# LeSEO - AI Content Refresh for WordPress

WordPress plugin that uses AI to refresh old blog posts. No new backlinks. No spam. No penalty risk.

## Installation

1. Download the [latest release](https://github.com/LeSEO-AI/LeSEO-Plugin/releases/latest)
2. Upload the `leseo-ai` folder to `/wp-content/plugins/`
3. Activate the plugin in WordPress admin
4. Go to **LeSEO > Settings** and enter your API key

## Features

- **Revive old posts** — AI rewrites decaying content while preserving your voice
- **Create new posts** — Generate SEO-optimized articles from topics
- **Voice matching** — Scans your last 10 posts to extract your writing style
- **Diff view** — See exactly what changed before publishing
- **Free plan** — 5 refreshes/month, no credit card required

## Requirements

- WordPress 6.0+
- PHP 7.4+
- API key from [leseo.app](https://leseo.app)
